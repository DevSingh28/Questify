SQL Intro

Definition: SQL (Structured Query Language) is a standardized programming language used to manage relational databases. It allows users to perform various operations such as querying, inserting, updating, and deleting data from databases.

Usage: SQL is widely used in database management systems like MySQL, PostgreSQL, SQLite, and Microsoft SQL Server for interacting with databases and manipulating data.

Benefits:

  • Provides a simple and powerful syntax for managing databases.
  • Allows for efficient data retrieval and manipulation.
  • Supports transactions to ensure data integrity.
  • Facilitates data analysis and reporting.

SQL Syntax

Example:

SELECT column1, column2
FROM table_name
WHERE condition;

Explanation: SQL statements typically follow a specific syntax. The SELECT statement is used to retrieve data from a database table. It specifies the columns to be retrieved from the table and can include a WHERE clause to filter the results based on specified conditions.

SQL SELECT

Example:

SELECT * FROM customers;

Explanation: The SELECT statement is used to retrieve data from one or more tables in a database. In this example, the wildcard (*) is used to select all columns from the "customers" table.

SQL WHERE

Example:

SELECT * FROM customers WHERE country = 'USA';

Explanation: The WHERE clause is used to filter records based on specified conditions. In this example, only the records with the country 'USA' will be selected from the "customers" table.

SQL ORDER BY

Example:

SELECT * FROM customers ORDER BY last_name;

Explanation: The ORDER BY clause is used to sort the result set in ascending or descending order. In this example, the records from the "customers" table will be sorted by the "last_name" column in ascending order.

SQL AND

Example:

SELECT * FROM customers WHERE country = 'USA' AND city = 'New York';

Explanation: The AND operator is used to combine multiple conditions in a WHERE clause. In this example, only the records with both 'USA' as the country and 'New York' as the city will be selected from the "customers" table.

SQL OR

Example:

SELECT * FROM customers WHERE country = 'USA' OR country = 'Canada';

Explanation: The OR operator is used to combine multiple conditions in a WHERE clause. In this example, records with either 'USA' or 'Canada' as the country will be selected from the "customers" table.

SQL NOT

Example:

SELECT * FROM customers WHERE NOT country = 'USA';

Explanation: The NOT operator is used to negate a condition in a WHERE clause. In this example, records with a country other than 'USA' will be selected from the "customers" table.

SQL Insert Into

Example:

INSERT INTO customers (first_name, last_name, email) VALUES ('John', 'Doe', 'john@example.com');

Explanation: The INSERT INTO statement is used to insert new records into a table. In this example, a new customer record with the specified first name, last name, and email address will be inserted into the "customers" table.

SQL CRUD Operations

Definition: CRUD stands for Create, Read, Update, and Delete. These are the four basic functions that can be performed on data in a database.

  • Create (INSERT): Adds new records to a table.
  • Read (SELECT): Retrieves data from a table.
  • Update (UPDATE): Modifies existing records in a table.
  • Delete (DELETE): Removes records from a table.

Examples:

1. Create (INSERT):

INSERT INTO customers (name, email) VALUES ('John Doe', 'john@example.com');

2. Read (SELECT):

SELECT * FROM customers;

3. Update (UPDATE):

UPDATE customers SET email='john.doe@example.com' WHERE id=1;

4. Delete (DELETE):

DELETE FROM customers WHERE id=1;

Explanation: These examples demonstrate how CRUD operations are performed in SQL. The INSERT statement adds a new customer to the "customers" table, the SELECT statement retrieves all customers, the UPDATE statement updates the email of a specific customer, and the DELETE statement deletes a customer record based on their ID.

SQL COUNT

Example:

SELECT COUNT(*) FROM customers;

Explanation: The COUNT function is used to return the number of rows that match a specified condition. In this example, it counts the total number of records in the "customers" table.

SQL SUM

Example:

SELECT SUM(order_amount) FROM orders;

Explanation: The SUM function is used to calculate the sum of values in a column. In this example, it calculates the total order amount from the "orders" table.

SQL Operation

Example:

SELECT (column1 * column2) AS result FROM table_name;

Explanation: SQL supports various mathematical and string operations on data. In this example, it calculates the product of values in column1 and column2 and aliases the result as 'result'.

SQL Inheritance

Definition: Inheritance in SQL refers to the ability of a child table (subclass) to inherit attributes and behaviors from a parent table (superclass).

Types of Inheritance:

  • Single Table Inheritance: All attributes of child classes are stored in a single table.
  • Class Table Inheritance: Each class in the inheritance hierarchy has its own table.
  • Concrete Table Inheritance: Each class in the inheritance hierarchy has its own table with all attributes.

Example:

CREATE TABLE employees (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    department VARCHAR(50)
);

CREATE TABLE managers (
    id INT PRIMARY KEY,
    name VARCHAR(50),
    department VARCHAR(50),
    manager_id INT,
    FOREIGN KEY (manager_id) REFERENCES employees(id)
);

Explanation: In this example, the "managers" table inherits attributes from the "employees" table. The "managers" table has an additional attribute "manager_id" to represent the manager's ID.
