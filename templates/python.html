What is Python?

Python is a popular programming language. It was created by Guido van Rossum, and released in 1991.

It is used for:

  • web development (server-side),
  • software development,
  • mathematics,
  • system scripting.

What can Python do?

  • Python can be used on a server to create web applications.
  • Python can be used alongside software to create workflows.
  • Python can connect to database systems. It can also read and modify files.
  • Python can be used to handle big data and perform complex mathematics.
  • Python can be used for rapid prototyping, or for production-ready software development.

Why Python?

  • Python works on different platforms (Windows, Mac, Linux, Raspberry Pi, etc).
  • Python has a simple syntax similar to the English language.
  • Python has syntax that allows developers to write programs with fewer lines than some other programming languages.
  • Python runs on an interpreter system, meaning that code can be executed as soon as it is written. This means that prototyping can be very quick.
  • Python can be treated in a procedural way, an object-oriented way or a functional way.

Good to know

  • The most recent major version of Python is Python 3, which we shall be using in this tutorial. However, Python 2, although not being updated with anything other than security updates, is still quite popular.
  • In this tutorial Python will be written in a text editor. It is possible to write Python in an Integrated Development Environment, such as Thonny, Pycharm, Netbeans or Eclipse which are particularly useful when managing larger collections of Python files.

Python Syntax compared to other programming languages

  • Python was designed for readability, and has some similarities to the English language with influence from mathematics.
  • Python uses new lines to complete a command, as opposed to other programming languages which often use semicolons or parentheses.
  • Python relies on indentation, using whitespace, to define scope; such as the scope of loops, functions and classes. Other programming languages often use curly-brackets for this purpose.

Python First Program

print("Hello, World!")

  • The traditional "Hello, World!" program is often the first program beginners write when learning a new programming language. It simply prints the string "Hello, World!" to the console, demonstrating the basic syntax for output in Python.
  • Function Call: The 'print()' function is used to display output in Python. It takes one or more arguments (separated by commas) and prints them to the standard output (usually the console).
  • String: The argument passed to print() is a string "Hello, World!". Strings in Python can be enclosed in single (') or double (") quotes.
  • Parentheses: Parentheses () are used to enclose the arguments passed to functions in Python. They are optional if the function doesn't take any arguments, but it's good practice to include them.
  • Quotation Marks: Quotation marks (") are used to denote strings in Python. Both single and double quotes are valid for defining strings, and you can use them interchangeably.
  • Whitespace: Python ignores leading and trailing whitespace around the string. However, whitespace inside the string is preserved and included in the output.

Python Comments

# This is a single-line comment
""" This is a multi-line comment """

  • Comments in Python are used to annotate code and provide explanations. Single-line comments start with a hash (#) symbol, while multi-line comments are enclosed in triple quotes (""" ... """). Comments are ignored by the Python interpreter and are useful for documenting code.

Python Variables

What are Variables? Variables in Python are used to store data values. A variable is like a container that holds a value which can be referenced and manipulated throughout the program.

Naming Rules: Variable names can contain letters, numbers, and underscores but cannot start with a number. They are case-sensitive (age, Age, and AGE are different variables). It's recommended to use descriptive names to make code more readable.

Dynamic Typing: Python is dynamically typed, meaning you don't need to specify the variable type explicitly. The interpreter automatically determines the data type based on the value assigned to the variable.



Common Types of Variables:

  • Integer (int): Stores whole numbers without decimal points.

    age = 25

  • Float (float): Stores floating-point numbers with decimal points.

    height = 5.9

  • String (str): Stores sequences of characters enclosed in single or double quotes.

    name = "John"

  • Boolean (bool): Stores either True or False.

    is_student = True

  • Examples:

    # Integer age = 25
    # Float height = 5.9
    # String name = "John"
    # Boolean is_student = True

Python Data Types

What are Data Types? Data types in Python represent the type of data that a variable can store. Python is dynamically typed, meaning you don't need to specify the data type explicitly; the interpreter automatically determines it based on the value assigned to the variable.

Common Data Types in Python:

  • Numeric Types:
    • Integer (int): Stores whole numbers without decimal points.

      x = 10 # Integer

    • Float (float): Stores floating-point numbers with decimal points.

      y = 3.14 # Float

    • Complex (complex): Stores complex numbers in the form 'a + bi'.

      z = 2 + 3j # Complex

  • Sequence Types:
    • List (list): Stores ordered collections of items. Lists are mutable, meaning their elements can be modified after creation.

      numbers = [1, 2, 3, 4, 5] # List

    • Tuple (tuple): Similar to lists, but tuples are immutable, meaning their elements cannot be changed after creation.

      point = (3, 4) # Tuple

    • Range (range): Represents a sequence of numbers.

      my_range = range(5) # Range

  • Mapping Type:
    • Dictionary (dict): Stores key-value pairs. Dictionaries are unordered collections of items, where each key maps to a corresponding value.

      x = 10 # Integer

  • Boolean (bool): Stores either True or False.

    is_student = True

Python Strings

Definition: A string in Python is a sequence of characters, enclosed in single ('') or double ("") quotes.

  • Characteristics:
    • Strings can contain letters, numbers, symbols, and whitespace.
    • Strings support various operations like indexing, slicing, concatenation, and formatting.
    • Strings are immutable, meaning their contents cannot be changed after creation.
  • Usage: Strings are used to represent text data in Python. They are fundamental data types and are widely used in various operations and applications, including text processing, data manipulation, and input/output operations.
  • Examples:
    • name = 'John'
    • message = "Hello, World!"
  • Operations:
    • Indexing: Access individual characters in a string using their position (index).
    • Slicing: Extract a substring from a string using slice notation.
    • Concatenation: Combine two or more strings into a single string using the + operator.
    • String Methods: Strings support various built-in methods for manipulating and formatting text, such as upper(), lower(), strip(), split(), join(), replace(), and more.
    • String Formatting: Format strings using the format() method or f-strings (formatted string literals) for more concise and readable formatting.

Python Lists

Definition: A list in Python is an ordered collection of items, enclosed in square brackets [].

  • Characteristics:
    • Lists can contain elements of different data types, including integers, floats, strings, and even other lists.
    • Lists are mutable, meaning their elements can be modified after creation.
    • List elements are ordered and can be accessed using indexing and slicing.
  • Usage: Lists are versatile and widely used in Python for storing and manipulating data. They are used to represent sequences of items, such as numbers, names, or any other type of data.
  • Examples:
    • numbers = [1, 2, 3, 4, 5]
    • names = ['John', 'Alice', 'Bob']
  • Operations:
    • Indexing: Access individual elements in a list using their position (index).
    • Slicing: Extract a sublist (slice) from a list using slice notation.
    • Appending and Extending: Add new elements to the end of a list using the append() method or add multiple elements using the extend() method.
    • Removing Elements: Remove elements from a list using methods like remove(), pop(), or del.

Python Tuples

Definition: A tuple in Python is an ordered collection of items, enclosed in parentheses ().

  • Characteristics:
    • Tuples are similar to lists but are immutable, meaning their elements cannot be changed after creation.
    • Tuples can contain elements of different data types, and they preserve the order of elements.
    • Tuple elements are accessed using indexing and slicing, similar to lists.
  • Usage: Tuples are often used to represent fixed collections of items, such as coordinates, database records, or function arguments.
  • Examples:
    • point = (3, 4)
    • coordinates = (x, y)
  • Operations:
    • Indexing: Access individual elements in a tuple using their position (index).
    • Slicing: Extract a subtuple (slice) from a tuple using slice notation.
    • Unpacking: Assign tuple elements to variables in a single line using tuple unpacking.

Python Dictionaries

Definition: A dictionary in Python is an unordered collection of key-value pairs, enclosed in curly braces {}.

  • Characteristics:
    • Dictionaries are mutable, meaning you can add, remove, or modify key-value pairs after creation.
    • Each key in a dictionary must be unique, and it maps to a corresponding value.
    • Dictionary keys can be of any immutable data type (e.g., string, integer, tuple), while values can be of any data type.
  • Usage: Dictionaries are commonly used for storing and retrieving data in a structured manner, such as user profiles, configuration settings, or any data that can be organized into key-value pairs.
  • Examples:
    • my_dict = {"name": "John", "age": 30}
    • empty_dict = {}
  • Operations:
    • Accessing Values: Use square brackets [] and the key to access the value associated with a key.
    • Adding or Modifying Elements: Assign a value to a new key or an existing key to add or modify elements in a dictionary.
    • Removing Elements: Use the del keyword, pop() method, or popitem() method to remove elements from a dictionary.
    • Dictionary Methods: Dictionaries support various methods for manipulating and accessing data, such as keys(), values(), items(), update(), clear(), and more.

Python Sets

Definition: A set in Python is an unordered collection of unique elements, enclosed in curly braces {}.

  • Characteristics:
    • Sets do not allow duplicate elements. If you try to add a duplicate element to a set, it will be ignored.
    • Sets are mutable, meaning you can add or remove elements from a set after its creation.
    • Set elements are unordered, meaning they are not indexed and their order may change arbitrarily.
  • Usage: Sets are useful for various operations involving distinct elements, such as finding unique items in a collection, performing set operations like union, intersection, and difference, and eliminating duplicates from a list or other data structures.
  • Examples:
    • my_set = {1, 2, 3, 4, 5}
    • my_set = set([1, 2, 2, 3, 3, 4]) # Removes duplicates
  • Operations:
    • Adding Elements: Use the add() method to add a single element to a set, and the update() method to add multiple elements.
    • Removing Elements: Use the remove() or discard() method to remove a specific element from a set.
    • Set Operations: Sets support various set operations like union (union()), intersection (intersection()), difference (difference()), symmetric difference (symmetric_difference()), and more.

Loops in Python

Definition: Loops in Python are control flow structures that execute a block of code repeatedly until a certain condition is met. They allow you to automate repetitive tasks and iterate over sequences of data.

Common Types of Loops

  • For Loop: Executes a block of code repeatedly for a specified number of times or over a sequence of items.

    Example:

    for i in range(5):
    print(i)

    Explanation: The for loop iterates over a sequence of numbers generated by the range function. In each iteration, the variable i takes on the value of the current number in the sequence, and the print statement displays it.

  • While Loop: Executes a block of code repeatedly as long as a specified condition is true.

    Example:

    count = 0
while count < 5:
    print(count)
    count += 1

    Explanation: The while loop continues to execute the block of code as long as the condition count < 5 is true. Inside the loop, the count variable is incremented by 1 in each iteration, and its value is printed.

Conditional Statements in Python

Definition: Conditional statements in Python are control flow structures that execute different blocks of code based on specified conditions. They allow you to make decisions in your code and control its flow based on various factors.

If-Else Conditions

  • If Statement: Executes a block of code if a specified condition is true.

    Example:

    x = 10
if x > 0:
    print("Positive")

    Explanation: The if statement checks whether the variable x is greater than 0. If the condition is true, the print statement inside the if block is executed, displaying "Positive".

  • If-Else Statement: Executes one block of code if a specified condition is true and another block if the condition is false.

    Example:

    x = -5
if x > 0:
    print("Positive")
else:
    print("Negative")

    Explanation: The if-else statement evaluates whether the variable x is greater than 0. If it is true, the first block of code (after the if keyword) is executed, displaying "Positive". Otherwise, the second block of code (after the else keyword) is executed, displaying "Negative".

  • If-Elif-Else Statement: Executes different blocks of code based on multiple conditions.

    Example:

    x = 0
if x > 0:
    print("Positive")
elif x < 0:
    print("Negative")
else:
    print("Zero")

    Explanation: The if-elif-else statement first checks if the variable x is greater than 0. If true, it displays "Positive". If not, it proceeds to the next condition and checks if x is less than 0. If true, it displays "Negative". If both conditions are false, it executes the else block and displays "Zero".

Python Functions

Definition: A function in Python is a reusable block of code that performs a specific task. Functions allow you to organize code into logical units, making it easier to understand, debug, and maintain.

  • Characteristics:
    • Functions can take input arguments, perform operations, and return results.
    • They can be defined using the def keyword followed by a function name, optional parameters, and a block of code.
    • Functions can have optional default parameters, variable-length argument lists, and keyword arguments.
  • Usage: Functions are used to encapsulate reusable code and perform specific tasks. They promote code reusability, modularity, and abstraction.
  • Examples:
    • def greet(name): print("Hello, " + name)
    • def add(x, y): return x + y

Python Lambda Functions

Definition: Lambda functions, also known as anonymous functions, are small, inline functions defined using the lambda keyword.

  • Characteristics:
    • Lambda functions can take any number of arguments but can only have one expression.
    • They are often used as short, throwaway functions for simple operations.
    • Lambda functions are typically used with higher-order functions like map(), filter(), and reduce().
  • Usage: Lambda functions are useful in situations where you need a small, temporary function without defining a separate named function.
  • Examples:
    • add = lambda x, y: x + y
    • square = lambda x: x ** 2

Python Try-Except Blocks

Definition: Try-except blocks in Python are used for exception handling, allowing you to handle errors and exceptions gracefully.

  • Characteristics:
    • The try block contains the code that may raise an exception.
    • The except block catches and handles exceptions that occur within the try block.
    • Multiple except blocks can be used to handle different types of exceptions.
    • An optional else block can be used to execute code if no exceptions occur in the try block.
    • An optional finally block can be used to execute cleanup code regardless of whether an exception occurs or not.
  • Usage: Try-except blocks are used to gracefully handle errors and prevent program crashes caused by unexpected exceptions.
  • Examples:
    • try: result = x / y except ZeroDivisionError: print("Error: Division by zero") else: print("Result:", result)
    • try: file = open("example.txt", "r") content = file.read() except FileNotFoundError: print("Error: File not found") finally: file.close()

Python Inheritance

Definition: Inheritance in Python is a mechanism where a class (subclass) inherits properties and behavior from another class (superclass). It allows you to create new classes based on existing ones, promoting code reuse and extensibility.

  • Characteristics:
    • Inheritance forms an "is-a" relationship between classes, where the subclass is a specialized version of the superclass.
    • Subclasses inherit attributes and methods from their superclass but can also define their own attributes and methods.
    • Python supports single inheritance (a subclass can inherit from only one superclass) and multiple inheritance (a subclass can inherit from multiple superclasses).
  • Usage: Inheritance is used to create hierarchical relationships between classes and promote code reuse by inheriting common functionality from a superclass.
  • Examples:
    • class Animal: def speak(self): print("Animal speaks") class Dog(Animal): def bark(self): print("Dog barks") class Cat(Animal): def meow(self): print("Cat meows") dog = Dog() dog.speak() # Output: Animal speaks dog.bark() # Output: Dog barks cat = Cat() cat.speak() # Output: Animal speaks cat.meow() # Output: Cat meows
  • Types of Inheritance:
    • Single Inheritance: A subclass inherits from only one superclass. This is the most common type of inheritance in Python.
    • Multiple Inheritance: A subclass inherits from multiple superclasses. Python supports multiple inheritance, allowing a subclass to inherit attributes and methods from more than one superclass.
    • Multilevel Inheritance: In multilevel inheritance, a subclass inherits from another subclass, creating a hierarchical chain of classes.
    • Hierarchical Inheritance: Hierarchical inheritance involves multiple subclasses inheriting from a single superclass. Each subclass can further have its own subclasses, forming a hierarchical structure.
    • Hybrid Inheritance: Hybrid inheritance is a combination of different types of inheritance, such as single inheritance, multiple inheritance, or multilevel inheritance.
