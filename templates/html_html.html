HTML Basics

Definition: HTML (Hypertext Markup Language) is the standard markup language for creating web pages. It defines the structure and content of a web page using a variety of tags and elements.

  • Characteristics:
    • HTML documents are comprised of a nested hierarchy of elements.
    • HTML tags are used to define the structure and semantics of a document.
    • HTML elements consist of a start tag, content, and an end tag, encapsulating the content to define its purpose.
  • Usage: HTML is used to create web pages that are rendered by web browsers. It allows developers to structure content, define headings, paragraphs, lists, links, images, tables, forms, and more.

HTML Basic Structure

Definition: The basic structure of an HTML document consists of several elements that define its overall layout and content. This structure provides a framework for organizing and presenting information on a web page.

  • Document Type Declaration (DOCTYPE): The DOCTYPE declaration specifies the version of HTML or XHTML used in the document. It is placed at the beginning of the document and helps browsers render the page correctly.
  • HTML Element: The <html> element is the root element of an HTML document and contains all other elements.
  • Head Section: The <head> section contains metadata about the document, such as the title, character set, and links to external resources like stylesheets and scripts.
  • Body Section: The <body> section contains the main content of the document, including text, images, links, forms, and other elements visible to users.

Demo Code:

        
            <!DOCTYPE html>
            <html lang="en">
            <head>
                <meta charset="UTF-8">
                <meta name="viewport" content="width=device-width, initial-scale=1.0">
                <title>My HTML Document</title>
            </head>
            <body>
                <header>
                    <h1>Welcome to My Website</h1>
                </header>
                <nav>
                    <ul>
                        <li><a href="#home">Home</a></li>
                        <li><a href="#about">About</a></li>
                        <li><a href="#contact">Contact</a></li>
                    </ul>
                </nav>
                <section id="home">
                    <h2>Home Section</h2>
                    <p>Welcome to my website! This is the home section.</p>
                </section>
                <section id="about">
                    <h2>About Section</h2>
                    <p>Learn more about us and our mission.</p>
                </section>
                <section id="contact">
                    <h2>Contact Section</h2>
                    <p>Contact us for inquiries and support.</p>
                </section>
                <footer>
                    <p>Copyright © 2024 My Website. All rights reserved.</p>
                </footer>
            </body>
            </html>

HTML Elements

Definition: HTML elements are the building blocks of web pages. Each element represents a specific type of content or functionality and is defined by tags enclosed in angle brackets.

  • Common Elements:
    • <h1> - <h6>: Headings of different levels, with <h1> being the highest level. Example: <h1>Main Heading</h1>
    • <p>: Paragraphs of text. Example: <p>This is a paragraph of text.</p>
    • <a>: Links to other web pages or resources. Example: <a href="https://www.example.com">Visit Example</a>
    • <img>: Images embedded within the document. Example: <img src="image.jpg" alt="Image">
    • <ul>: Unordered lists. Example: 
                              
                            <ul>
                                <li>Item 1</li>
                                <li>Item 2</li>
                                <li>Item 3</li>
                            </ul>
    • <ol>: Ordered lists. Example: 
                              
                            <ol>
                                <li>First Item</li>
                                <li>Second Item</li>
                                <li>Third Item</li>
                            </ol>
    • <li>: List items within <ul> or <ol>. Example: 
                              
                            <ul>
                                <li>Item 1</li>
                                <li>Item 2</li>
                            </ul>
    • <div>: Division or section of the document, used for layout and styling purposes. Example: <div id="content">Content Goes Here</div>
    • <span>: Inline container for text or other inline elements, used for styling purposes. Example: <span style="color: red;">Red Text</span>
  • Attributes: HTML elements can have attributes that provide additional information about the element or modify its behavior. Common attributes include id, class, href, src, alt, style, etc.

HTML Forms

Definition: HTML forms are used to collect user input and submit it to a web server for processing. They consist of input fields, buttons, and other elements for gathering data from users.

  • Common Form Elements:
    • <form>: Defines a form for user input.
    • <input>: Represents an input field, such as text input, checkbox, radio button, etc.
    • <textarea>: Defines a multi-line text input control.
    • <select>: Creates a drop-down list or combo box.
    • <button>: Defines a clickable button.
  • Form Attributes: HTML form elements can have various attributes such as action, method, name, placeholder, value, etc.

HTML Tables

Definition: HTML tables are used to display data in a tabular format with rows and columns. They consist of <table> element containing one or more <tr> (table row) elements, which in turn contain <td> (table data) or <th> (table header) elements.

  • Common Table Elements:
    • <table>: Defines a table.
    • <tr>: Defines a row in a table.
    • <td>: Defines a cell in a table.
    • <th>: Defines a header cell in a table.
  • Table Attributes: HTML table elements can have attributes such as border, cellpadding, cellspacing, width, height, etc.

Example:

        
            <table border="1" cellpadding="5" cellspacing="0">
                <tr>
                    <th>Name</th>
                    <th>Age</th>
                    <th>Gender</th>
                </tr>
                <tr>
                    <td>John</td>
                    <td>25</td>
                    <td>Male</td>
                </tr>
                <tr>
                    <td>Emily</td>
                    <td>30</td>
                    <td>Female</td>
                </tr>
            </table>

HTML Metadata

Definition: HTML metadata elements provide information about the document, such as its title, character set, author, and viewport settings. They are typically placed within the <head> section of an HTML document.

  • Common Metadata Elements:
    • <title>: Defines the title of the document, displayed in the browser's title bar or tab.
    • <meta>: Provides metadata about the HTML document, such as character set, viewport settings, author, description, etc.
    • <link>: Defines a relationship between the current document and an external resource, such as a stylesheet or icon.
    • <base>: Specifies the base URL and target for all relative URLs within the document.
  • Viewport Meta Tag: The <meta name="viewport" content="width=device-width, initial-scale=1.0"> tag is commonly used to control the layout and scaling of web pages on mobile devices.

HTML Links

Definition: HTML links are used to navigate between web pages or resources. They are created using the <a> (anchor) element, which defines a hyperlink to another location specified by the href attribute.

  • Anchor Element: The <a> element is used to create hyperlinks. It has the following attributes:
    • href: Specifies the URL of the page or resource to link to.
    • target: Specifies where to open the linked document. Common values include _blank (opens in a new window), _self (opens in the same frame), etc.
    • title: Provides additional information about the link, displayed as a tooltip when the user hovers over it.
  • Linking to CSS: To link an external CSS stylesheet to an HTML document, use the <link> element within the <head> section. Example: <link rel="stylesheet" type="text/css" href="styles.css">
  • Linking to JavaScript: To link an external JavaScript file to an HTML document, use the <script> element with the src attribute. Example: <script src="script.js"></script>

HTML Semantic Elements

Definition: HTML5 introduced semantic elements that provide a more meaningful structure to web documents. These elements convey the purpose or meaning of the content, making it easier for search engines and assistive technologies to understand.

  • Common Semantic Elements:
    • <header>: Defines a header for a document or section.
    • <nav>: Defines navigation links within a document.
    • <main>: Defines the main content area of a document.
    • <section>: Defines a section of content within a document.
    • <article>: Defines an independent piece of content, such as a blog post or article.
    • <aside>: Defines content aside from the main content, such as sidebars or related links.
    • <footer>: Defines a footer for a document or section.
  • Benefits: Semantic elements improve accessibility, search engine optimization (SEO), and maintainability of web pages by providing clearer structure and meaning to the content.

HTML vs. HTML5

Differences: HTML5 introduced several new features and improvements over previous versions of HTML, including:

  • Semantic Elements: HTML5 introduced semantic elements such as <header>, <footer>, <nav>, and <article> to provide better structure and meaning to web documents.
  • Improved Support for Multimedia: HTML5 includes native support for audio and video elements, allowing developers to embed media content directly into web pages without relying on third-party plugins like Flash.
  • Form Enhancements: HTML5 introduced new input types, attributes, and validation features for creating more interactive and user-friendly web forms. Examples include email, url, number, date, and required.
  • Canvas and SVG: HTML5 introduced the <canvas> element for drawing graphics and animations dynamically using JavaScript, as well as the <svg> element for scalable vector graphics.
  • Local Storage: HTML5 introduced the localStorage and sessionStorage APIs for storing data locally in the user's browser, enabling web applications to cache data and improve performance.
  • Geolocation: HTML5 introduced the Geolocation API for accessing the user's geographical location through the browser, enabling location-aware web applications and services.

Compatibility: HTML5 is backward compatible with older versions of HTML, meaning that existing HTML documents should still render correctly in modern browsers. However, to take advantage of HTML5 features, developers need to use the appropriate syntax and elements supported by HTML5-compliant browsers.
